Future Wealth Starter Program

Free Financial Literacy for Schools + Family Financial Onboarding System

What This Program Does

We provide free financial literacy education in primary schools and give parents access to optional financial services through licensed banks using consent-based referrals.

How It Works

  1. Schools receive free program
  2. Learners take workbooks home
  3. Parents attend workshops
  4. Parents optionally register interest
  5. Leads go to licensed financial institutions
